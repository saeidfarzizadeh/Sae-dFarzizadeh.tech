## مسئله

محیط‌های صنعتی به ادراک کم‌تأخیر و قابل‌اعتماد در گره‌های سنسور توزیع‌شده نیاز دارند. پردازش متمرکز گلوگاه ایجاد می‌کند؛ استنتاج ساده لبه نیز توان را روی محاسبات تکراری هدر می‌دهد.

## معماری

سامانه از خط لوله فیوژن سلسله‌مراتبی استفاده می‌کند: گره‌های محلی استخراج‌کننده‌های سبک اجرا می‌کنند و هماهنگ‌کننده فقط رویدادهای پراهمیت را تجمیع می‌کند.

## نکات پیاده‌سازی

```rust
pub struct FusionNode {
    imu: ImuDriver,
    model: TfliteInterpreter,
}

impl FusionNode {
    pub fn tick(&mut self) -> Option<Event> {
        let frame = self.imu.sample()?;
        let score = self.model.predict(&frame);
        (score > self.threshold).then(|| Event::from(frame, score))
    }
}
```
