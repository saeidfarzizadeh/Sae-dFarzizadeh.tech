## Problem

Endüstriyel ortamlar, dağıtık sensör düğümlerinde güvenilir ve düşük gecikmeli algılamaya ihtiyaç duyar. Merkezi işleme darboğaz yaratır; basit kenar çıkarımı ise gereksiz hesaplamayla güç israf eder.

## Mimari

Sistem hiyerarşik bir füzyon hattı kullanır: yerel düğümler hafif özellik çıkarıcıları çalıştırır, koordinatör yalnızca yüksek önemli olayları toplar.

## Uygulama Öne Çıkanları

```rust
pub struct FusionNode {
    imu: ImuDriver,
    model: TfliteInterpreter,
}

impl FusionNode {
    pub fn tick(&mut self) -> Option<Event> {
        let frame = self.imu.sample()?;
        let score = self.model.predict(&frame);
        (score > self.threshold).then(|| Event::from(frame, score))
    }
}
```

## Sonuçlar

| Metrik | Önce | Sonra |
| ------ | ---- | ----- |
| Gecikme (p95) | 45ms | 2.1ms |
| Güç tüketimi | 3.2W | 0.9W |
| Yanlış pozitif | 12% | 3% |
