## Problem

Industrial environments need reliable, low-latency perception across distributed sensor nodes. Centralized processing introduces bottlenecks; naive edge inference wastes power on redundant computation.

## Architecture

The system uses a hierarchical fusion pipeline: local nodes run lightweight feature extractors, while a coordinator aggregates only high-salience events.

## Implementation Highlights

```rust
pub struct FusionNode {
    imu: ImuDriver,
    model: TfliteInterpreter,
}

impl FusionNode {
    pub fn tick(&mut self) -> Option<Event> {
        let frame = self.imu.sample()?;
        let score = self.model.predict(&frame);
        (score > self.threshold).then(|| Event::from(frame, score))
    }
}
```
